# 🩺 Cough Analysis System

A beginner-friendly Python application that helps users identify whether their cough is more likely to be **dry** or **wet (productive)** based on selected symptoms. The program also provides educational information on home remedies, common over-the-counter medicines, and alerts users when symptoms suggest they should seek urgent medical attention.

> **Disclaimer:** This application is for educational purposes only. It is **not** intended to diagnose, treat, cure, or prevent any disease and should not replace advice from a qualified healthcare professional.

---

## Features

- Simple menu-based symptom selection
- Distinguishes between dry and wet coughs
- Provides home care recommendations
- Suggests common over-the-counter medications
- Detects emergency warning symptoms
- Easy to understand and beginner-friendly Python code

---

## Symptoms Included

- Dry cough
- Wet cough (producing mucus/phlegm)
- Scratchy throat
- Chest congestion
- Runny nose
- Fever
- Difficulty breathing
- Chest pain
- Blood when coughing
- Persistent cough (more than 3 weeks)

## How to Run

### Requirements

- Python 3.10 or later

### Steps

1. Clone the repository

```bash
git clone https://github.com/yourusername/cough-analysis-system.git
```

2. Navigate to the project folder

```bash
cd cough-analysis-system
```

3. Run the program

```bash
python cough_analysis.py
```
